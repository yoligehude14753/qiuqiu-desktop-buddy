# 球球 · 电脑搭子（Qiuqiu Desktop Buddy）

一个 macOS 桌面悬浮数字人：它会按固定频率看一眼你的屏幕，看懂你在干嘛，然后用足球老炮儿的腔调给你贫两句——写代码、刷网页、看球都陪。底层的视觉理解与语音合成服务**已全部内置**，你不需要配置任何模型或 API。

## 下载与使用

1. 到 [Releases](../../releases) 下载最新的 `球球电脑搭子.dmg`（或 `.app`）。
2. 打开后首次会弹出设置面板，在「激活码」里填入作者发放给你的激活码（向作者私下领取，不在本仓库公开）。
3. 点「激活」，再点右上角绿色播放按钮即可开始陪看。
4. 首次启动若提示，请到「系统设置 → 隐私与安全性 → 屏幕录制」勾选本应用并重开。

## 特性

- 看屏理解：办公 / 阅读 / 浏览 / 视频 / 球赛等场景自动识别。
- 足球魂逗哏：看到啥都能用足球梗砸挂，看球时进解说模式。
- 中 / 英双语，可在设置切换（英文模式输出英文解说）。
- 四种口音人设：北京老炮 / 上海爷叔 / 山东大汉 / 东北老铁。
- CosyVoice 语音播报，声画同到。
- 鼠标穿透、窗口召回、暂停 / 退出、历史统计。

## 快捷键

- `Cmd+Shift+K`：切换鼠标穿透
- `Cmd+Shift+P`：开始 / 暂停
- `Cmd+Shift+J`：把窗口召回到主屏右上角

## 隐私

运行时会截取屏幕缩略图用于画面理解。处理敏感资料时请暂停或退出。激活码与配置只保存在本机 `~/Library/Application Support/qiuqiu-desktop-buddy/`。
